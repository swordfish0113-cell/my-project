# 法律文书起草样式选择说明

使用本 skill 起草文书时，先确认文书场景，再读取 `references/` 中对应的官方样式文件：

| 场景 | 优先参考 |
| --- | --- |
| 自然人作为原告起诉 | `references/court-civil-procedure/civil-complaint-individual.md` |
| 法人或其他组织作为原告起诉 | `references/court-civil-procedure/civil-complaint-organization.md` |
| 自然人答辩 | `references/court-civil-procedure/civil-answer-individual.md` |
| 提起民事上诉 | `references/court-civil-procedure/civil-appeal.md` |
| 诉前或仲裁前财产保全 | `references/court-civil-procedure/property-preservation-pre-litigation.md` |
| 诉中财产保全 | `references/court-civil-procedure/property-preservation-litigation.md` |
| 诉讼代理授权 | `references/court-civil-procedure/power-of-attorney-individual.md` 或 `power-of-attorney-organization.md` |
| 申请强制执行 | `references/court-civil-procedure/enforcement-application.md` |
| 送达地址确认 | `references/court-civil-procedure/service-address-confirmation.md` |

不要把本文件当作模板正文引用；实际文书结构以官方样式文件为准。
